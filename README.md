📌 What It Is
Zippy is a macOS AppleScript-based utility app designed to:
Perform batch media conversions (like MP4, MP3, MOV, PNG)


Extract media streams (separating audio/video)


Download videos and files using popular CLI tools


It uses macOS’s built-in UI prompts to create an interactive tool for non-technical users, while running powerful command-line tools in the background.

💾 Installation
Extract the zip file.


Move the Zippy app from the extracted folder into the Applications folder.


When you open the Zippy app for the first time, macOS will show a warning.


Go to:
 System Settings → Privacy & Security → Allow apps from Anywhere


At the bottom, you’ll see an "Open Anyway" option for the Zippy app — click it.
